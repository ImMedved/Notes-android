# About
- This is an app for displaying and editing text notes and timers.

- Notes that support md elements are now available.

- Timers and stopwatches that count from a specific date or to a specific date are now available, so they support yearly countdown ranges and work even when the app and device are turned off.

- Synchronization with other clients via the server is now available.
