# Android-клиент

## Что уже реализовано

- markdown notes с editor/preview режимами
- timers и stopwatch-сценарии
- светлая и темная expressive-тема
- локальное хранение через Room
- sync settings через DataStore
- ручная и фоновая синхронизация через WorkManager
- home screen widgets для заметки и таймера

## UI-концепция

Светлая тема:

- градиент `white -> beige -> soft lilac`
- светлые paper-like поверхности
- мягкий редакторский характер

Темная тема:

- градиент `black -> wine red`
- золотые акценты
- более драматичный контраст для таймеров и sync-состояний

Типографика собрана в `neo-gothic / editorial` духе:

- display и section headlines на выразительной serif-основе
- body-текст на читаемом sans-serif

## Локальные данные

Локально клиент хранит:

- заметки
- таймеры
- статус синхронизации
- настройки сервера
- режим темы

## Поведение синхронизации

Клиент работает по серверному контракту:

1. Локально сохраняет изменения.
2. Пушит pending-операции.
3. Повторно получает полный `snapshot`.

Это соответствует текущей backend-модели без delta sync.

## Виджеты

Виджеты читают уже не файловый cache, а актуальные данные из repository/Room.

Они поддерживают:

- refresh
- запуск приложения в нужный раздел
- toggle/reset для timer widget
